Class: KssParameter

kss.KssParameter

A KssParameter object represents a single parameter of a KssSection.

This class is normally accessed via the kss module:

const KssParameter = require('kss').KssParameter;

Constructor

new KssParameter(dataopt)

Creates a KssParameter object and stores the given data.

If passed an object, it will add section, name, and description properties.
